docker-ci

[![Build Status](https://travis-ci.org/cakebake/docker-ci.svg?branch=master)](https://travis-ci.org/cakebake/docker-ci)

PHP 7.2
-------

-	PHP 7.2
	-	with the ionCube PHP Loader (enabled)
	-	with Zend OPcache
	-	with Xdebug
-	Composer
-	Node.js
-	NPM
-	Less
-	Sass
-	wget
-	curl
-	mariadb-client
-	zip & unzip
-	git
-	openssh-client
-	rsync

### Docker Pull

```bash
docker pull cakebake/ci:php7.2
```
